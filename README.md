# Heart_Disease_Prediction
## Problem Statement :
This project, Heart Disease Prediction, analyzes patient data to identify key risk factors such as age, BMI, smoking, diabetes, family history, and lifestyle habits, and predicts whether a person is likely to have heart disease or not using Exploratory Data Analysis (EDA) and visualizations.

## Dataset Link:
https://www.kaggle.com/datasets/oktayrdeki/heart-disease
